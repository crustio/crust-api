# API

## Queries

### Chain info

#### Get header info

Request:

```shell
curl GET 'http://localhost:56666/api/v1/block/header'
```

Response:

```json
{
    "number": 13871,
    "hash": "0xeeadb2bd71c125c98ce0892d830931fb27c771ab0025c60735fca9419abae990"
}
```

#### Get system health

Request:

```shell
curl GET 'http://localhost:56666/api/v1/system/health
```

Response:

```json
{
    "peers": 19,
    "isSyncing": false,
    "shouldHavePeers": true
}
```

#### Get block hash with block number

Request:

```shell
curl GET 'http://localhost:56666/api/v1/block/hash?blockNumber=148589
```

Response:

```json
"0xd006c6597bed500f99c5e0b943f597a0ede4d0ecb96b671a029221267c169a40"
```
